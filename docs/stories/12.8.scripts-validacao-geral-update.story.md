# 12.8 — Scripts de Validação Geral e Update

## Executor Assignment

```yaml
executor: "@dev"
quality_gate: "@qa"
quality_gate_tools: [code_review, integration_test]
```

## Story

**As a** operador do deployer,
**I want** scripts de validação geral e update,
**so that** eu possa verificar o estado completo da instalação e atualizar configs quando necessário.

## Description

Criar dois scripts:
- `deployer/scripts/validate.sh` — validação completa: state files, configs, gateway, skills, workspace
- `deployer/scripts/update.sh` — backup + git pull + re-gera configs + re-valida

## Acceptance Criteria

- [ ] AC1: **validate.sh:** Verifica state files existem, configs válidos, gateway acessível, skills instalados, workspace presente
- [ ] AC2: **update.sh:** Backup configs atuais, git pull deployer, re-gera configs, re-valida
- [ ] AC3: Ambos usam libs do deployer para output consistente (ui.sh, logger.sh)
- [ ] AC4: validate.sh retorna exit code 0 se tudo OK, 1 se algum FAIL
- [ ] AC5: update.sh faz backup antes de qualquer modificação

## Scope

### IN
- `deployer/scripts/validate.sh`
- `deployer/scripts/update.sh`
- Output via ui.sh (step_ok/step_fail/step_skip)

### OUT
- Não executa ferramentas do deployer (apenas valida/atualiza)
- Não modifica state files

## Tasks / Subtasks

- [ ] **Task 1: Criar `deployer/scripts/validate.sh`**
  - [ ] 1.1 Check state files (dados_whitelabel, dados_openclaw, dados_workspace, etc.)
  - [ ] 1.2 Check configs válidos (via validate-config.sh)
  - [ ] 1.3 Check gateway acessível (curl /health)
  - [ ] 1.4 Check skills instalados
  - [ ] 1.5 Check workspace presente
  - [ ] 1.6 Resumo final

- [ ] **Task 2: Criar `deployer/scripts/update.sh`**
  - [ ] 2.1 Backup configs atuais
  - [ ] 2.2 git pull deployer
  - [ ] 2.3 Re-gerar configs (chamar validate-config.sh)
  - [ ] 2.4 Re-validar via validate.sh

- [ ] **Task 3: Testes**
  - [ ] 3.1 Testar validate.sh com instalação completa
  - [ ] 3.2 Testar update.sh com backup funcional

**Dependência:** Story 12.7 (validate-config.sh)

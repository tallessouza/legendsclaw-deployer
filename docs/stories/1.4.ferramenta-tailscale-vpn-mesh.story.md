## Story

**As a** operador,
**I want** conectar minha VPS (ou maquina local) ao Tailscale mesh via deployer interativo,
**so that** possa acessar o gateway e servicos de forma segura sem expor portas publicas.

## Acceptance Criteria

1. Instala Tailscale via script oficial (`curl -fsSL https://tailscale.com/install.sh | sh`)
2. Coleta hostname Tailscale (input com loop confirmado)
3. Autentica e exibe link de auth (abre `tailscale up --hostname=$hostname`)
4. Hints de setup local (Windows/Mac/WSL2) para conectar desktop ao mesh
5. Opcionalmente habilita Tailscale Funnel para HTTPS publico
6. Salva em `~/dados_vps/dados_tailscale`
7. Verifica conectividade (`tailscale status`)
8. Logs salvos em `~/legendsclaw-logs/tailscale-{timestamp}.log`

```bash
tailscale status

# Obter IP Tailscale
tailscale ip -4
```

[Source: docs/referencia/whitelabel-install-guide.md#3.2]

### Dual-Mode: Local vs VPS

Tailscale funciona em ambos os modos. A deteccao de ambiente (`detectar_ambiente()` de `deployer/lib/env-detect.sh`) pode ser usada para personalizar hints:
- **VPS**: Hints focados em conectar desktop ao mesh, Funnel para HTTPS publico
- **Local**: Hints focados em conectar a outras maquinas no mesh, Funnel para expor servicos locais

A instalacao via curl funciona igual em ambos os modos.

[Source: docs/architecture/legendsclaw-architecture.md#5-arquitetura-de-infraestrutura]

### Topologia de Rede com Tailscale

```

  Claude Code                                        Docker Swarm
  Bridge.js                                          Traefik, Portainer

```

**Decisao:** Tailscale mesh network.
**Motivo:**
- Zero config de firewall/NAT
- WireGuard nativo (criptografia moderna)
- Funnel para HTTPS publico sem Cloudflare

[Source: docs/architecture/legendsclaw-architecture.md#14-decisoes-arquiteturais]

### 8-Step Lifecycle para 04-tailscale.sh

1. RESOURCE GATE — verificar `curl` disponivel (Tailscale e leve, sem gate CPU/RAM)
2. LOAD STATE — `dados()`
3. INPUT COLLECTION — hostname Tailscale (loop confirmado)
4. RESOLVE DEPENDENCIES — verificar se Tailscale ja instalado → skip se sim
5. INSTALL — `curl -fsSL https://tailscale.com/install.sh | sh` (retry 3x)
6. AUTHENTICATE — `tailscale up --hostname=$hostname` + aguardar auth
7. VERIFY — `tailscale status` + opcionalmente Funnel
8. FINALIZE — salvar dados_tailscale + hints desktop + resumo

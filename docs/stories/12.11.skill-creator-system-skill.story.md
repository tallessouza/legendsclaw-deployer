# 12.11 — Skill Creator (System Skill)

## Executor Assignment

```yaml
executor: "@dev"
quality_gate: "@architect"
quality_gate_tools: [code_review, pattern_validation, skill_validation]
```

## Story

**As a** desenvolvedor do agente,
**I want** um system skill que scaffolde novas skills automaticamente,
**so that** eu possa criar novas skills com a estrutura correta (SKILL.md + index.js + tools/).

## Description

Criar um "skill creator" skill na categoria system que gera a estrutura completa de uma nova skill: SKILL.md (documentação), index.js (entry point com handler e health), tools/ (diretório para tools). Registra automaticamente no skills/index.js.

## Acceptance Criteria

- [ ] AC1: Skill scaffolda nova skill com SKILL.md + index.js + tools/
- [ ] AC2: Registra no skills/index.js automaticamente
- [ ] AC3: Funciona via comando do agente no gateway
- [ ] AC4: SKILL.md gerado segue formato padrão (description, tier, env vars, usage)
- [ ] AC5: index.js gerado tem handler + health placeholder

## Scope

### IN
- `deployer/apps/_template/skills/system/skill-creator/SKILL.md`
- `deployer/apps/_template/skills/system/skill-creator/index.js`
- Template de scaffolding para novas skills

### OUT
- Não implementa UI para criação de skills
- Não publica skills externamente

## Dev Notes

### Estrutura Gerada por Skill

```
skills/{category}/{skill-name}/
├── SKILL.md        # Documentação
├── index.js        # Entry point (handler + health)
└── tools/          # Tool implementations
    └── .gitkeep
```

### SKILL.md Template

```markdown
# {skill-name}

## Description
{description}

## LLM Tier
{tier}

## Environment Variables
None required

## Usage
\`\`\`javascript
const skill = require('./{skill-name}');
const result = await skill.handler({ /* input */ });
const health = await skill.health();
\`\`\`
```

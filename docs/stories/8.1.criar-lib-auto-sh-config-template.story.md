## Story

**As a** operador que precisa testar o deployer de ponta a ponta sem intervencao manual,
**I want** uma biblioteca `lib/auto.sh` com funcao `input()` que substitui `read -rp` e um template de configuracao com todas as chaves,
**so that** as ferramentas possam operar em modo automatizado (AUTO_MODE) lendo inputs de um arquivo de config, mantendo o modo interativo inalterado.

## Acceptance Criteria

1. `deployer/lib/auto.sh` criado com 3 funcoes: `auto_load_config()`, `input()`, `auto_confirm()`
2. `input "key" "prompt" var` usa Bash nameref (`local -n`) para atribuir ao caller — requer Bash 4.3+
3. Em `AUTO_MODE=true`, `input()` busca valor da chave no associative array `_AUTO_VALUES`
4. Em modo normal (`AUTO_MODE=false`), `input()` chama `read -rp` (comportamento identico ao atual)
5. Flag `--secret` usa `read -rsp` em modo interativo e redact no log em AUTO_MODE
6. Flag `--required` faz `input()` retornar exit 1 se chave ausente em AUTO_MODE
7. Flag `--default=X` preenche valor default se vazio (tanto em auto quanto interativo)
8. `auto_confirm()` retorna "s" em AUTO_MODE, chama `read -rp` em modo normal
9. `auto_load_config()` parseia config file (formato `key: value`, ignora `#` e linhas vazias) para associative array
10. `deployer/auto-config.example` criado com todas as ~40 chaves organizadas por ferramenta, com comentarios explicativos
11. Testes BATS em `tests/deployer/lib-auto.bats` cobrem: load config, input lookup, auto_confirm, --secret, --required, --default, chave ausente, modo interativo fallback

## Dev Notes

### Arquitetura da Solucao

A funcao `input()` usa **Bash 4.3+ nameref** (`local -n _var_ref="$3"`) para atribuir diretamente na variavel do caller, mantendo o mesmo padrao de `read -rp "Prompt: " var`.

**Pattern de transformacao (para stories 8.2/8.3):**
```bash
# ANTES:
read -rp "Dominio do Portainer (ex: painel.exemplo.com): " dominio_portainer

# DEPOIS:
input "base.dominio_portainer" "Dominio do Portainer (ex: painel.exemplo.com): " dominio_portainer --required
```

**Internals de `input()`:**
- Recebe: config_key, prompt, variable_name, [flags]
- Em AUTO_MODE: `_var_ref="${_AUTO_VALUES[$config_key]:-$default_val}"`
- Em modo normal: `read -rp "$prompt" _var_ref` (ou `read -rsp` se --secret)
- Nameref collision evitada com prefix `_var_ref` / `_confirm_ref`

**Config file format:**
```
# Comentario
base.dominio_portainer: painel.exemplo.com
base.pass_portainer: MinhaSenh4Segura!
```
- Parse via `while IFS= read -r line` + awk split em `: `
- Associative array `declare -gA _AUTO_VALUES`

## Story

**As a** operador que instala o Legendsclaw via deployer,
**I want** um visual premium com ASCII art em block characters e paleta Gold/Yellow,
**so that** o instalador transmita profissionalismo e identidade visual consistente com a marca Legendsclaw.

## Acceptance Criteria

1. `deployer/lib/tui.sh` criado com paleta de cores Gold/Yellow como cor primaria
2. Constantes de cor: `TUI_GOLD` (Bold Yellow `\033[1;33m`), `TUI_WHITE`, `TUI_GREEN`, `TUI_RED`, `TUI_GRAY`, `TUI_DIM`, `TUI_CYAN`, `TUI_NC`, `TUI_BOLD`
3. `tui_logo()` imprime banner "LEGENDSCLAW" em block chars (`██╗` style) na cor Gold
4. `tui_menu_header()` imprime banner "DEPLOY" em block chars com borda Gold
5. `tui_ferramenta_header "$num" "$nome"` imprime banner "INSTALANDO" generico com numero/nome da ferramenta
6. `tui_concluido()` imprime banner "CONCLUIDO" em verde com borda Gold
7. `tui_separator()` imprime linha horizontal Gold (`═══════════`)
8. Todas as funcoes fazem reset de cor com `TUI_NC` ao final
9. BATS tests em `tests/deployer/lib-tui.bats` validam que cada funcao produz output com block chars esperados (`██`, `═══`)
10. Largura maxima de 60 caracteres de conteudo imprimivel (cabe em terminal 80 colunas)

### Referencia: SetupOrion ASCII Art Pattern

[Source: `docs/referencia/orion-scripts/SetupOrion.sh:282-298`]
```bash
nome_instalador() {
    clear
    echo ""
    echo -e "$branco       ███████╗███████╗████████╗██╗   ██╗██████╗ ...
    # 6 linhas de block chars formando "SETUP ORION"
    # Seguido por versao "2.8.0"
}
```
O Orion usa `$branco` (white) para block chars e `$amarelo` (yellow) para bordas.
Legendsclaw inverte: `TUI_GOLD` para block chars (mais impactante), `TUI_WHITE` para texto secundario.

### Paleta de Cores Completa

```bash
# Cor primaria da marca — Gold/Yellow
readonly TUI_GOLD='\033[1;33m'

# Cores funcionais
readonly TUI_WHITE='\033[1;37m'
readonly TUI_GREEN='\033[0;32m'
readonly TUI_RED='\033[0;31m'
readonly TUI_GRAY='\033[0;90m'
readonly TUI_DIM='\033[2m'
readonly TUI_CYAN='\033[0;36m'

# Controle
readonly TUI_NC='\033[0m'
readonly TUI_BOLD='\033[1m'
```

### Pattern BATS (seguir para TODOS os tests)

[Source: `tests/deployer/lib-ui.bats` — pattern existente]

```bash
#!/usr/bin/env bats

LIB_DIR="$(cd "$(dirname "$BATS_TEST_FILENAME")/../../deployer/lib" && pwd)"

setup() {
  TEST_DIR=$(mktemp -d)
  source <(sed 's/^readonly //g' "$LIB_DIR/tui.sh")
}

teardown() {
  rm -rf "$TEST_DIR" 2>/dev/null || true
}

@test "tui_logo produz output com block chars" {
  local output
  output=$(tui_logo)
  [[ "$output" == *"██"* ]]
}

@test "tui_ferramenta_header inclui numero e nome" {
  local output
  output=$(tui_ferramenta_header "01" "Traefik")
  [[ "$output" == *"[01]"* ]]
  [[ "$output" == *"Traefik"* ]]
}
```

### Largura de Referencia

- Terminal minimo: 60 colunas
- Terminal padrao: 80 colunas
- O ASCII art LEGENDSCLAW ocupa ~60 chars — cabe em 80 col com margem
- O ASCII art DEPLOY ocupa ~49 chars — cabe confortavelmente
- Se "INSTALANDO" ultrapassar 60 chars, usar "INSTALL" (49 chars)
- Se "CONCLUIDO" ultrapassar 60 chars, usar "DONE" ou remover acentos

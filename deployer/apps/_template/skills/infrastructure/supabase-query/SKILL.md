# Supabase Query

Execute database queries against Supabase via REST API. Supports
select, insert, update, and RPC calls.

## Capabilities
- Execute database queries via REST
- Insert and update records
- Call stored procedures (RPC)

## Configuration
- Requires: SUPABASE_URL, SUPABASE_ANON_KEY, SUPABASE_SERVICE_ROLE_KEY

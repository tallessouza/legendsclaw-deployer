# ClickUp Ops

ClickUp task management integration. Create, update, and query tasks
and workspaces via the ClickUp API.

## Capabilities
- Create and update tasks
- Query workspace and team data
- Manage task assignments and statuses

## Configuration
- Requires: CLICKUP_API_KEY, CLICKUP_TEAM_ID

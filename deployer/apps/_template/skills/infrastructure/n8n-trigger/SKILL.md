# N8N Trigger

Triggers N8N automation workflows via webhook. Dispatch workflows
with parameters and monitor execution status.

## Capabilities
- Trigger workflows via webhook
- Pass parameters to workflow execution
- Monitor workflow execution status

## Configuration
- Requires: N8N_API_KEY, N8N_WEBHOOK_URL

# Planner Skill

Decomposes complex tasks into actionable steps. Plans execution
order, identifies dependencies, and tracks progress.

## Capabilities
- Decompose complex tasks into subtasks
- Identify task dependencies and ordering
- Track execution progress
- Suggest optimal execution strategies

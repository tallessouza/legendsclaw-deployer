# Orchestration Skills

Skills for managing and dispatching multiple tasks.

## Included Skills

| Skill | Description |
|-------|-------------|
| planner | High-level project planning (prefer superpowers/writing-plans for code) |
| task-orchestrator | Manage and dispatch multiple independent tasks |

## Task Orchestrator

Use when you have many independent subtasks that can run in parallel:
- Large refactoring projects
- Multi-file changes
- Independent test suites
- Parallel data processing

## Planner

General-purpose planning skill. For code-specific planning, prefer the
superpowers/writing-plans skill which is optimized for development tasks.
